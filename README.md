## Overview
**Swift.ai** is a SaaS platform that leverages AI to generate content and images efficiently. The platform allows users to create articles, blog titles, resumes, and AI-generated images while providing premium feature access control. It aims to reduce manual effort in content creation and enhance productivity.

---

## Features
- **AI-Powered Content Generation:** Generate articles, blog titles, and resume content using OpenAI API.  
- **AI Image Manipulation:** Text-to-image generation, background removal, and object removal via Clipdrop API, delivered with Cloudinary.  
- **Secure Authentication:** User accounts managed via Clerk with PostgreSQL storage for user-specific content.  
- **Responsive UI/UX:** Built with React (Vite + Tailwind CSS) for a modern, mobile-first interface.  
- **Premium Feature Access:** Advanced AI features are gated based on subscription plans.

---

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** PostgreSQL  
- **Authentication:** Clerk  
- **APIs:** OpenAI API, Clipdrop API  
- **Cloud Storage:** Cloudinary

## Installation & Setup
1. Clone the repository:  
```bash
git clone https://github.com/Shruti1suman/SwiftAI.git
```
2. Navigate to the project directory:
```bash
cd SwiftAI
```
3. Install dependencies for frontend and backend:
```bash
npm install
```
4. Run the development server:
```bash
npm run dev
```
5. Open http://localhost:3000 to view the app.
